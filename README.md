# iSite Research Engine Starter

This repository is a starter scaffold for building a Codex-backed iSite research engine for country-level high-value property scanning and **DIS (digital indoor system)** recommendation workflows.

## Included

- Codex project scaffolding
- iSite references and skill files
- configuration templates
- public-source adapter placeholders
- core evidence / validation / proxy standardization modules
- test scaffolding

## Remote unit terminology

- **pRRU**: pure indoor environments
- **hRRU**: semi-open environments

Use **DIS** as the standard term instead of `digital DAS`.
